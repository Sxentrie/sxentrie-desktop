## **Strict Naming Conventions**

- **Directories:** Use kebab-case (e.g., local-data, feature-modules).
- **React Components:** Use PascalCase.tsx (e.g., ChatThread.tsx, TaskGraph.tsx). This signals that the file exports a React functional component.
- **TypeScript/Node Logic:** Use camelCase.ts (e.g., vectorSearch.ts, shellExecutor.ts).
- **Zod Schemas:** Suffix with Schema (e.g., chatPayloadSchema.ts) to easily distinguish them from pure TypeScript type definitions.
